Web Scraping

El web scraping es una técnica para poder sacar información de un sitio web. Lo bueno del web scraping es que lo haces rápidamente y puedes elegir que elementos quieres descargar.

Se puede hacer con muchos lenguajes de programación, aunque, lo que recomiendo personalmente es Python porque es un lenguaje sencillo y se puede hacer cualquier cosa con él.

También podemos scrapear páginas con muchos otros lenguajes como por ejemplo, Java, o C# incluso con powershell de una forma potente.

En el vídeo adjuntado en la derecha, explico como funciona un programa que scrapea una página web y a partir de los datos que obtenemos ejecuta diferentes acciones.

El web scraping en definitiva sirve para obtener datos de una página web. Y por ejemplo, se puede utilizar para comprar proudctos en amazon, o hacer una auditoría de el tipo de información que hay, o cualquier cosa que se te ocurra que puedas hacer a través de la obtención de datos en una página web.
